이 프로그램은 java를 이용해 작성된 Library Management System 입니다

jar 파일이 있는 폴더에서 powerShell 창을 열고
 java -jar Library.jar 
입력하시면 실행됩니다.

기능은 종류별 책 보유 현황 보기, 책 대출/예약, 책 반납, 사용자별 대출 현황, 회원 정보 수정 이 있습니다.
각 기능의 번호를 입력하면 각 기능들을 사용할 수 있습니다.
프로그램 종류를 원하시면 6을 입력하시면 됩니다. 

1. 종류별 책 보유 현황 보기
도서관에 보유한 책 중 도서 장르별로 몇권이 있는 지 확인 할 수 있습니다.
input 값으로 넣을 수 있는 종류는 
 '문학','종교', '순수과학', '기술과학', 사회과학', '예술', '역사','총류', '철학','언어' 가 있습니다.

2. 책 대출/예약
책을 대출하고 책이 대출중이면 예약할 수 있는 시스템입니다.
도서관에 같은 책은 없고, 예약은 책 1권 당 1명만 가능합니다.
도서를 대출, 예약한 사람이 없으면 대출이 가능합니다.책이 대출중이고 예약자가 없으면 예약 가능합니다. 책이 반납된 상태고 예약중이면 예약자만 대출가능합니다. 
input 값으로 '대출자의 member_id', 대출 할 도서의 제목을 입력합니다. 


3. 책 반납
대출한 책을 반납할 수 있습니다.
input 값으로 반납자의 member_id, 대출할 도서의 제목을 입력합니다.

4.  사용자별 대출 현황
member id를 입력하면 회원이 대출한 책들을 출력합니다.

5. 회원정보 수정
회원의 이메일과 전화번호를 변경할 수 있습니다.
member_id를 입력하고 새로운 이메일, 전화번호를 입력하면 회원정보가 수정됩니다.
